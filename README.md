 var –This is the old way of declaring variables, used before ES6. However, its features are that it can be reassign and redeclare.it is bad habbit .

    - 3️⃣ What is the difference between map(), filter(), and forEach()?
ANS:
forEach() – goes through each item and does something, but does not create a new array.

map() – goes through each item, changes it, and returns a new array.It mainly change all array

filter() – goes through each item, keeps only the ones that match a condition, and returns a new array.
