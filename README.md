Ans:Difference between var, let, and const

4. What is an arrow function?
Arrow function হলো JavaScript এ ছোট করে function লেখার একটি উপায়।
5. What are template literals?
Template literals দিয়ে backtick ( ) ব্যবহার করে string লেখা হয় এবং এর মধ্যে variable সহজে ব্যবহার করা যায়।
